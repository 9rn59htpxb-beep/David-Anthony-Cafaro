# David-Anthony-Cafaro

Control repo for David Cafaro's GitHub work coordinated with AI agents.

## Quick start
- Review `loopback/manifest.yaml` to see tracked repositories and their roles.
- Read `loopback/agent-playbook.md` for rules agents follow when making changes.
- When you need work done, copy `loopback/tasks/TEMPLATE.md` to a new file in `loopback/tasks/`, fill it out, and open an issue that links to it.

## What lives here
- `loopback/` — control docs, repository map, and task templates
- `loopback/repo-map.md` — human-readable inventory derived from the manifest
- `loopback/tasks/` — task intake files tied to GitHub issues

## How to get help quickly
- Open a GitHub issue with a short description of the request.
- Add a task file under `loopback/tasks/` with acceptance criteria and links to any relevant repos or branches.
- Mention any constraints (timelines, branches to avoid, or approvals needed).
